# Gestor de Notas con Citas APA y Multimedia

Este sistema permite:
- Capturar notas clasificadas
- Registrar citas APA
- Subir multimedia (fotos, videos, audio)
- Guardar en SQLite
- Exportar como PDF y ZIP
